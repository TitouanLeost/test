\doxysubsection{Detailed Description}
A class to handle the \mbox{\hyperlink{classGPS}{GPS}}. 

This class allow to get the coordinates, the course and the speed of the boat. Moreover, it contains a method to get the number of satellites used by the \mbox{\hyperlink{classGPS}{GPS}} and two methods to get the date and the time (usefull for logging data).
